What data is recorded, how is it used, and how long is it stored?

Technical characteristics

When using Seanox XMEX Service, no data is collected, logged or analyzed by the service.

The service only creates completely anonymous statistics of usage. This contains: number of requests, incoming data volume, outgoing data volume, sum of processor time of all requests and number of occurred errors.

By using third-party service providers for the provision of the network infrastructure, we only have influence on the service endpoint for which Seanox Software Solutions is responsible.

Data storage of their transmitted data

One of the characteristics of Seanox XMEX Service is its use as a data store. The user is explicitly responsible for this data.

Seanox XMEX service provides volatile storages. The data is marked for deletion after about 15 minutes of non-use.The deletion of the data is then triggered by other incoming request.

Are social-media plug-ins and links to social media platforms used?

This is not used and should not be used. With one exception, links to the project on https://github.com are used.
